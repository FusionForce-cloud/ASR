# Wav2Vec2-XLSR Telugu ASR

This repository provides code to fine-tune **Wav2Vec2-XLSR (facebook/wav2vec2-xls-r-300m)** on the AI4Bharat Kathbath Telugu ASR dataset. It includes data preprocessing, training, evaluation, and a Streamlit app for inference.

## Setup

1. **Clone the repository** and navigate to its directory.
2. **Install dependencies:**  
   ```bash
   pip install -r requirements.txt
